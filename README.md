# Система записи в салон красоты

Десктопное приложение для автоматизации работы администратора салона красоты. Позволяет управлять записями с проверкой доступности времени и вести историю посещений. Разработано на Python с использованием графического интерфейса Tkinter.

# Основные возможности:

Управление клиентской базой: добавление, редактирование, поиск клиентов

Каталог услуг с привязкой к специализации мастеров

Управление расписанием мастеров

Интеллектуальная система записи с поиском свободных временных слотов

Динамическая фильтрация мастеров по выбранной услуге

Валидация всех вводимых данных

Ведение истории посещений для каждого клиента

Графический интерфейс с вкладками для разных функций

# Технологии:

Язык программирования: Python 3.10

Графический интерфейс: Tkinter

Работа с датами и временем: модуль datetime

Архитектура: объектно-ориентированное программирование (ООП)

Система контроля версий: Git

# Установка и запуск:

Предварительные требования:
Установленный Python версии 3.8 или выше.

1. Клонирование репозитория:

git clone https://github.com/ninnafil/beauty_salon.git
cd beauty_salon

2. Запуск приложения:

python main.py
